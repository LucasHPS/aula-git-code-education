# Como foi realizado o Push #

O Push deste conteúdo foi realizado após a criação deste arquivo **README**, utilizando os seguintes comandos:

1. Commit :
  > git commit -a -m "Inclusão do arquivo README.md"

2. Push :
  > git push origin master
